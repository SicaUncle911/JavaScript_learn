原始数据类型

Undefined

意为未定义，仅有一个值undefined，且是一个假值，在var及let声明未初始化值时则自动赋值undefined。
使用typeof无法区分是声明了变量但没初始化值还是没有声明变量但使用typeof检测类型，因为它们都会返回undefined。

Null

意为空，仅有一个值null，且是一个假值，表示一个空指针对象。
当某变量需要保存对象但目前没对象保存 就可用null填充。

Boolean

拥有字面值true和false，可用Boolean()函数将其他数据类型转换为Boolean型，if语句的条件判断自动转换Boolean型。

Number

Number类型的字面值有整数值和浮点值以及NaN，超出范围用Infinity和-Infinity表无穷。

String

表示0或多个16位Unicode字符序列。

Symbol

值是唯一不可变的符号，作用是确保对象属性使用唯一标识符。

复杂数据类型

Object

一组数据和功能的集合，通过new 对象名创建。使用instanceof判定
